TaskFlow is a simple and user-friendly task management web application built to help users organize their daily tasks efficiently. This project was developed as part of my learning journey and is suitable for college submissions, internships, and placement portfolios.

The main goal of TaskFlow is to provide a clean interface where users can create, manage, and track their tasks easily without any complexity.


Key Features

Secure Authentication - Sign up and login with JWT-based security
Task Management - Create, read, update, and delete tasks effortlessly
Smart Search - Find tasks quickly by title or description
Filter & Sort - Organize by status, priority, and more
User Profiles - Personalize your account settings
Responsive Design - Works beautifully on all devices
Beautiful UI - Warm, human-friendly design with smooth animations

Tech Stack
Frontend

React 18 - Modern UI library
React Router - Seamless navigation
TailwindCSS - Beautiful, responsive styling
Axios - HTTP requests
Vite - Lightning-fast development

Backend

Node.js - JavaScript runtime
Express - Web framework
MongoDB - NoSQL database
Mongoose - MongoDB object modeling
JWT - Secure authentication
Bcrypt - Password hashing

Using the Application
First Time Setup

Register an Account

Click "Get Started" or "Register"
Fill in your name, email, and password
Password must contain uppercase, lowercase, and numbers
Click "Create Account"


Create Your First Task

Click "Add New Task"
Enter task details (title, description, priority, due date)
Click "Create Task"


Manage Your Tasks

Edit - Click the edit button to update task details
Delete - Remove tasks you no longer need
Search - Use the search bar to find specific tasks
Filter - Filter by status (pending/in-progress/completed) or priority


Update Your Profile

Click "Profile" in the navigation
Update your name, bio, and avatar
Click "Save Changes"
